# Dr. Rishi Dhar — Medical Landing Page

Premium single-page medical landing page for **Dr. Rishi Dhar**, Chest Medicine & Respiratory Specialist based in Barasat, West Bengal.

## 🚀 Tech Stack
- **React** (Vite)
- **Tailwind CSS v4** (via @tailwindcss/vite)
- **Google Maps Embed** for clinic location
- **WhatsApp** direct booking integration

## 📋 Features
- ✅ Fixed navbar with pulse-animated "Book Appointment" CTA
- ✅ Hero section with dark premium design
- ✅ Trust/social proof strip
- ✅ 3-card Services grid with hover effects
- ✅ About section with stats
- ✅ Chamber hours table + Google Maps embed
- ✅ Floating WhatsApp FAB with bounce animation
- ✅ Fully mobile-responsive
- ✅ SEO meta tags

## 🏃 Getting Started
```bash
npm install
npm run dev
```

## 🏗 Build for Production
```bash
npm run build
```
